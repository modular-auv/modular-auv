# Modular AUV
This is an SDP (Student Directed Project) by Matthew Baltay '28 and Kyle Benton '28

Coached by Mr. Muschamp (a.k.a. MUGSY)

## What is the point of our project?
Organizations that do exploration, sample gathering, and other underwater applications shouldn't have to buy subs with features they don't need. We strive to create a modular, easy-to-use, and useful framework for modular AUVs.
